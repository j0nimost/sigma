## SIGMA
An Interpreter / REPL written in C#. Get the grammar definition [here](./Grammar.md) 

```
Sigma > 
```


[![Build Status](https://travis-ci.com/j0nimost/sigma.svg?branch=main)](https://app.travis-ci.com/j0nimost/sigma)
